Build a cinematic romantic birthday website called “A Film Made For Mohammad”.

This is not a normal birthday landing page. It should feel like a private short film website made by Melody for Mohammad. The user has real videos of themselves and Mohammad, including videos from before they met in person, so the whole website should feel like an emotional movie timeline: before meeting, first connection, memories, voiceover, birthday ending.

Use React + Vite.
Use Framer Motion for animations.
Use a clean component structure.
Make it mobile-first and beautiful on desktop.

Overall concept:
The website should behave like a romantic film that unfolds while scrolling.

Structure:
- Opening title
- Prologue: before we met
- Scene 1
- Scene 2
- Voiceover interlude
- Memory montage
- Eight frames / reasons
- Secret scene
- Final scene / birthday ending
- End credits

Visual style:
- Deep burgundy, black cherry, warm gold, ivory text.
- Cinematic film grain.
- Soft vignette.
- Subtle gold light leaks.
- Elegant typography.
- No childish hearts, no cheesy glitter, no generic Valentine design.
- It should feel like an intimate film, candlelight, old cinema, velvet, handwritten love, and archived memories.

Important UX:
- The site should feel like watching a film through scrolling.
- Each section is a “scene”.
- Add a fixed top scroll progress bar styled like a film timeline.
- Add a small fixed “sound on / sound off” button.
- Background music should NOT autoplay with sound. It should only play after user clicks.
- Use smooth scene transitions: fade to black, film burn, soft light leak, slow zoom.
- Add subtitles/captions under video placeholders.
- Video should be lazy-loaded and never break if missing.
- If an asset is missing, show a beautiful cinematic placeholder frame instead.
- Respect prefers-reduced-motion.
- Avoid horizontal overflow.
- Mobile experience must be excellent.
