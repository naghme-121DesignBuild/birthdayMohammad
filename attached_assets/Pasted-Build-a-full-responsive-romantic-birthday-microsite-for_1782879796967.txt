Build a full responsive romantic birthday microsite for my partner Mohammad.

Project concept:
This is not a generic birthday landing page. It should feel like a private digital love letter that unfolds like a cinematic story. The emotional direction is: elegant, intimate, warm, deep burgundy, gold, soft light, quiet luxury, personal, not cheesy, not childish.

Use React + Vite. Use Framer Motion for animations if possible. Use clean component structure. The site must be mobile-first and also look beautiful on desktop.

Visual style:
- Deep burgundy / wine background: #2b0310, #3a0715, #4a0d1c
- Gold accents: #c9a227, #d8b94c
- Ivory text: #fff7ec
- Soft blush shadows: rgba(255, 190, 190, 0.08)
- Elegant serif typography for headings, readable serif/sans for body.
- Use Google Fonts:
  - Playfair Display for main headings
  - Cormorant Garamond or Libre Baskerville for body
  - Great Vibes or Parisienne only for small accent phrases, not body text
- Add subtle grain/noise overlay.
- Add very subtle floating gold dust particles or soft glowing orbs in the background.
- Do not use bright red hearts, childish icons, glitter overload, or cheap Valentine style.
- The page should feel like velvet, candlelight, a sealed letter, and a private night.

Important UX:
- Mobile-first.
- Smooth scrolling.
- Every section should have breathing space.
- No huge walls of text.
- Use scroll reveal animations.
- Respect prefers-reduced-motion.
- Do not autoplay audio with sound. Add a music toggle button.
- Add placeholders for videos, images, voice notes, and background music.
- All video placeholders should be easy to replace from /public/assets/videos.
- All image placeholders should be easy to replace from /public/assets/images.
- Add comments in code showing where I should replace video/image/audio files.
